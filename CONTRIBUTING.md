### 1. Adding to the Archive
If you have digital memorabilia, audio, or art:
* Create a new folder under `/archives` (e.g., `/archives/art/`).
* Upload your file with a clear, descriptive name.
* Include a small `.txt` file with any metadata (date, location, source) if possible.

### 2. Improving Documentation
Help us keep our README and Roadmap up to date! If you see a typo or have a suggestion for a new milestone, feel free to open a Pull Request.
